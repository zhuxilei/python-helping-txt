### 2.输出 Output
print作为Python中最常用的一种输出方式将print后面的内容输出到屏幕。
print会将它后面的表达式当做字符串来输出并自动加入换行，当我们相接多个表达式时，只要在表达式间加上逗号就可以了。
```markdown
print "I am", 24, "years old"
```
